# PiggyBankYield

A simple yield-sharing PiggyBank smart contract built with Solidity 0.8.20.

## Features

- Users can deposit ETH
- Shares are issued per deposit
- Admin can add yield
- Users can withdraw proportional share

## Tech

- Solidity 0.8.20
- Foundry (forge, cast) for deployment and testing
- Anvil as local blockchain
